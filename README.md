# Website

## Creation

Documentation of the steps to create this app.

```bash
# from apps/
# see https://www.npmjs.com/package/create-svelte
pnpm create svelte@latest
```

## Developing

Once you've installed packages for the entire repo with `pnpm install`, start a development server:

```bash
turbo dev
```

## Building

To create a production version of your app:

```bash
turbo build
```

You can preview the production build with `turbo preview`.
